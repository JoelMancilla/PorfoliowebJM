Sobre Mi

Soy un entusiasta del diseño y la tecnología: Diseñador de Interfaces y Experiencia de Usuario con formación técnica en Programación.

Mi pasión está en innovar y bocetar ideas para dar vida a interfaces de productos digitales que no solo sean visualmente atractivas, sino también intuitivas y funcionales. Creo que el núcleo de un gran producto está en las personas, por lo que estudio constantemente sus reacciones y comportamientos al interactuar con él, transformando esas observaciones en insights valiosos. Estoy en una búsqueda activa de mejorar mis conocimientos día a día, fusionando la perspectiva creativa del diseño con el pensamiento lógico del desarrollo para crear soluciones digitales coherentes, viables y centradas en el usuario.

Gracias por visitar mi pequeño rincón de la web. ¡Estoy emocionado de compartir más de mi carrera contigo!

Agro XP

AgroXP es un proyecto que realize en la prueba final de mi carrera. Se trata de una pagina web agronomo con el fin de mostrar los servicios y productos agricolas, tambien un formato de contacto simple. La pagina tiene un deafio de cambiar el diseño cotidiano de (marron, verde), este proyecto fue desarrollado con las tecnologias como Figma y Photoshop para el diseño (Html, Css, Javascript) Para el desarrollo

Instituto SSC Frames

Instituto Superior San Carlos, Es la realizacion de un prototipo web sobre un instituto cuyas funcionalidades pricipales son las inscripciones a travez de formularios y compras de cursos. estre prototipo fue vendido a un Programador De la Universidad Nacional de Cordoba. este proyecto fue desarrollado con las tecnologias como (Figma, Miro, Photoshop)

Web Space

Web Space es un pequeño proyecto sobre la creacion de un menu diferente al resto, Este menu se presenta con Opciones que llevan fotografias o iconos esparcidos sobre la pantalla, y estan diseñado para pequeñas paginas web con el propocito de tener opciones mas faciles de reconocer. este proyecto fue desarrollado con las tecnologias como (Figma, Illustrator) y (Html Css y Javascript)
